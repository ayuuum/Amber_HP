# CTA自動最適化ルール

## 使用方法
1. 記事の内容を読み取り、記事タイプを判定
2. 記事タイプに応じて最適なCTAを自動選択
3. 推奨CTAタイプと日本語CTA文言3案を出力

---

## 記事タイプ判定フローチャート

```
記事の内容を確認
    ↓
【キーワードに「比較」「選び方」「おすすめ」が含まれる】
    ↓ YES
→ 記事タイプ：比較・検討記事
→ 推奨CTA：30分無料壁打ち相談
    ↓ NO
【キーワードに「導入」「始める」「始め方」が含まれる】
    ↓ YES
→ 記事タイプ：導入直前記事
→ 推奨CTA：事例ヒアリング
    ↓ NO
【キーワードに「方法」「やり方」「チェックリスト」が含まれる】
    ↓ YES
→ 記事タイプ：情報収集記事
→ 推奨CTA：無料チェックリストDL
    ↓ NO
→ 記事タイプ：その他
→ 推奨CTA：無料相談（汎用）
```

## 記事タイプ別CTA選択ルール

### 1. 情報収集記事

**判定条件：**
- キーワードに「方法」「やり方」「チェックリスト」「手順」が含まれる
- 記事内容が「知りたい」「学びたい」という情報収集意図
- 具体的な手順・方法を説明している

**推奨CTAタイプ：** 無料チェックリストDL

**なぜ最適か：** 情報収集段階の読者は、まず資料をダウンロードして検討したい。強引な相談依頼は離脱を招く。

**CTA文言案（3案）：**

#### 案1：シンプル型
```
【無料ダウンロード】AI導入失敗を防ぐチェックリスト

中小企業のAI導入で失敗しないための10項目チェックリストを無料で提供しています。
導入前に必ず確認すべきポイントをまとめました。
```

#### 案2：ベネフィット強調型
```
AI導入前に確認すべき10項目を無料でチェック

「本当にAI導入が必要か？」「予算は適切か？」など、
導入前に確認すべき10項目をチェックリスト形式でまとめました。
今すぐ無料ダウンロードして、失敗を防ぎましょう。
```

#### 案3：緊迫感型
```
【今すぐ無料DL】AI導入失敗を防ぐ10項目チェックリスト

中小企業のAI導入失敗率は約70%と言われています。
このチェックリストで、失敗のリスクを大幅に減らせます。
今すぐ無料でダウンロードしてください。
```

### 2. 比較・検討記事

**判定条件：**
- キーワードに「比較」「選び方」「おすすめ」「違い」が含まれる
- 記事内容が「どれを選ぶか」「何が良いか」という比較検討意図
- 複数の選択肢を比較している

**推奨CTAタイプ：** 30分無料壁打ち相談

**なぜ最適か：** 比較検討段階の読者は、専門家の意見を聞きたい。相談という形で自然にCTAを挿入できる。

**CTA文言案（3案）：**

#### 案1：シンプル型
```
【無料相談】AI導入の選び方、専門家に相談しませんか？

「どのAIツールが自社に合うか分からない」
「導入費用の目安を知りたい」
そんな悩みを30分の無料相談で解決します。
```

#### 案2：ベネフィット強調型
```
AI導入の選び方で迷っているなら、無料相談がおすすめ

AI導入には「自社の課題に合った選び方」が重要です。
30分の無料相談で、あなたの会社に最適なAI導入方法を提案します。
相談は完全無料。まずは気軽にお問い合わせください。
```

#### 案3：緊迫感型
```
【今すぐ無料相談】AI導入の選び方を間違えると失敗します

AI導入の失敗率は約70%。その多くは「選び方のミス」が原因です。
30分の無料相談で、失敗を防ぐ選び方をアドバイスします。
今すぐお問い合わせください。
```

### 3. 導入直前記事

**判定条件：**
- キーワードに「導入」「始める」「始め方」「導入事例」が含まれる
- 記事内容が「実際に導入する」「導入を決めた」という導入直前意図
- 具体的な導入方法・事例を説明している

**推奨CTAタイプ：** 事例ヒアリング

**なぜ最適か：** 導入直前の読者は、成功事例や具体的な導入プロセスを知りたい。事例ヒアリングという形で自然にCTAを挿入できる。

**CTA文言案（3案）：**

```

```
AI導入の成功事例を無料でヒアリング。失敗を防ぐポイントが分かります

実際にAI導入を成功させた企業の事例を、無料でヒアリングできます。
「導入前の課題」「導入プロセス」「得られた効果」を詳しくお話しします。
成功事例を知ることで、あなたの導入も成功に近づきます。
```

#### 案3：緊迫感型
```
【今すぐ無料ヒアリング】AI導入の成功事例を知れば、失敗を防げます

AI導入の失敗率は約70%。成功事例を知ることで、失敗のリスクを大幅に減らせます。

## CTA挿入のベストプラクティス

### 挿入位置

1. **記事中盤（H2の後）**
   - 読者が記事の価値を理解した後
   - 自然な流れでCTAを挿入

2. **記事終盤（最後のH2の後）**
   - 記事の内容を理解した後
   - 次のアクションを促す

3. **サイドバー（記事横）**
   - 記事を読みながら常に見える位置
   - 離脱を防ぐ

### 避けるべき挿入位置

- 記事冒頭（読者が価値を理解する前）
- 記事中盤の連続したCTA（押し付けがましい）
- 記事終盤の複数CTA（選択肢が多すぎる）

### CTAデザインのポイント

- **色：** Amberのカラーパレット（Deep Forest Green #1F3326）を使用
- **サイズ：** 目立つが、記事の邪魔をしないサイズ
- **文言：** 簡潔で、行動を促す明確な表現
- **アイコン：** ダウンロード→下矢印、相談→メッセージアイコンなど
