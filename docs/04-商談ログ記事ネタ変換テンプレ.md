# 商談ログ記事ネタ変換テンプレ

## 使用方法
1. 商談メモを下記のフォームに入力
2. よくある誤解・不安、検索キーワード、記事タイトル案を自動生成
3. 見出し構成と一次情報要素を整理

---

## 入力フォーム

### 商談基本情報

| 項目 | 内容 |
|------|------|
| **商談日** | YYYY-MM-DD |
| **企業規模** | 従業員数 / 業種 |
| **商談担当者** | 役職 / 名前 |
| **商談時間** | 分 |

### 商談メモ（箇条書きで記入）

```
1. 
2. 
3. 
4. 
5. 
```

### 記事タイトル案（3案）

商談メモから、読者の課題を解決する記事タイトルを3案作成します。

#### タイトル案1：課題解決型
```
【キーワード】で【課題】を解決する【方法】
```

**例：**
```
AI導入で中小企業の業務効率化を実現する5つの方法
```

#### タイトル案2：誤解解消型
```
【誤解】は間違い？【正しい理解】と【実践方法】
```

**例：**
```
「AI導入は高額」は間違い？中小企業でも予算内で導入する方法
```

#### タイトル案3：事例・実績型
```
【成功事例】から学ぶ【課題解決】の【ポイント】
```

**例：**
```
中小企業のAI導入成功事例から学ぶ、失敗を防ぐ5つのポイント
```

## 商談ログから記事ネタを抽出するコツ

### 1. 課題・不安の抽出
- 「〜が不安」「〜が心配」という表現に注目
- 「〜が分からない」「〜が疑問」という表現に注目
- 過去の失敗経験の言及に注目

### 2. 誤解の抽出
- 「〜だと思っていた」「〜だと思っていたが」という表現に注目
- 「〜は無理では」「〜は難しいのでは」という表現に注目
- 一般的な誤解の言及に注目

### 3. 検索キーワードの抽出
- 商談で出た「課題の言い回し」をそのままキーワード化
- 「中小企業」「AI導入」などの組み合わせ
- 実際に検索されそうな自然な言い回し

### 4. 一次情報の抽出
- Amberの実績・事例を商談メモと関連付ける
- 現場経験から得た知見を商談メモと関連付ける
- 商談ログから得た課題・誤解を一次情報として活用
