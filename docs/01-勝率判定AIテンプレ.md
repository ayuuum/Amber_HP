# 勝率判定AIテンプレ

## 使用方法
1. 下記の表にキーワードと上位3記事の情報を入力
2. 各評価項目を0-100点で採点
3. ウェイトを掛けて合計スコアを算出

## スコアリング基準（参考）

### 検索意図マッチ度（0-100点）
- 90-100点：Amberのサービスが直接的に解決する課題
- 70-89点：Amberのサービスが間接的に解決する課題
- 50-69点：関連はあるが、直接的な解決ではない
- 0-49点：Amberのサービスと関連が薄い

### 競合の弱さ（0-100点）
- 90-100点：上位記事が全て一般論・抽象論、一次情報ゼロ
- 70-89点：一部に一次情報があるが、実務レベルではない
- 50-69点：一次情報はあるが、Amberが上回れる内容
- 0-49点：一次情報が豊富で、Amberが勝てる可能性が低い

### 一次情報優位性（0-100点）
- 90-100点：現場経験・実績・商談ログを大量に活用できる
- 70-89点：一部の一次情報を活用できる
- 50-69点：一次情報はあるが、競合と差別化しにくい
- 0-49点：一次情報が少なく、一般論になりがち

### LLMO引用適性（0-100点）
- 90-100点：定義文・箇条書き・数値・テンプレが全て揃う
- 70-89点：3-4項目が揃っている
- 50-69点：1-2項目が揃っている
- 0-49点：LLMOが引用しにくい構造

### CV導線の作りやすさ（0-100点）
- 90-100点：自然にCTAを挿入できる箇所が複数ある
- 70-89点：1-2箇所にCTAを挿入できる
- 50-69点：無理やりCTAを入れる必要がある
- 0-49点：CTAを入れると不自然になる

## 使用例
